# Architecture Decision Records

This folder captures significant design decisions for the Unlock app. Each ADR answers:

- **Context** — what problem are we solving
- **Decision** — what we chose
- **Rationale** — why
- **Consequences** — what this commits us to

ADRs are immutable once accepted. When a decision is revisited or overridden, add a new ADR that supersedes the prior one — don't edit in place.
