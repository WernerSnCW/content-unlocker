## Step 1 — Create PRODUCT_GOALS.md in the project root

Save the following file to the project root as PRODUCT_GOALS.md.
This file is the source of truth for the strategic assessment and
will be used by the 060 prompt on every future run automatically.

```markdown
# PRODUCT GOALS
## Unlock Document Intelligence Tool

---

## What this tool is for

The Unlock Document Intelligence Tool is a content management and
recommendation system for Unlock's founding investor sales campaign.
It manages a registry of investor-facing documents, analyses call
transcripts to detect investor signals, recommends the right document
to send at each stage of the pipeline, generates new content using
AI with built-in compliance checking, and identifies gaps in content
coverage across investor archetypes and pipeline stages.

---

## Primary user

Tom King and Unlock sales agents conducting the founding investor
outreach campaign. Users are sophisticated but non-technical — they
need the tool to surface the right action with minimal friction.
The tool must be fast enough to use between calls and clear enough
to use without training.

---

## Success criteria

- The correct document is recommended for the correct investor
  archetype at the correct pipeline stage, every time
- Generated content passes QC on the first attempt without requiring
  manual fixes for compliance language or figure framing
- Content gaps across the 3 archetype × 5 stage coverage matrix
  are identified and briefed automatically before they cause a
  failed recommendation
- Aircall transcripts from the 30,000-contact outreach campaign
  can be uploaded in batches, analysed for persona and readiness,
  and linked to leads without manual data entry
- Every investor-facing document in the registry complies with
  Content Bank V4 rules — correct figures, approved terminology,
  no prohibited language — before it reaches an investor
- When Unlock's product or compliance context changes, affected
  documents are identified and flagged for review automatically
  before the next campaign session

---

## Out of scope

- This tool is not a CRM — lead management is secondary to
  document intelligence
- This tool does not replace Pipedrive or Aircall — it reads
  from them (via transcript upload) but does not write back
- This tool is not a publishing platform — documents are managed
  here but delivered via email outside the tool
- This tool is not for investors — it is for the sales team only
- General-purpose AI writing assistance outside the Unlock
  compliance framework is explicitly out of scope

---

## Version

Last updated: April 2026
```

Confirm the file is saved before proceeding to Step 2.
